\# AI Interview Preparation Assistant



\## Project Overview



AI Interview Preparation Assistant is a Python and Streamlit-based web application that helps candidates prepare for interviews by generating role-based interview questions and evaluating their responses.



\## Features



\* Enter any job role to generate interview questions

\* Submit answers through the web interface

\* Automatic basic answer scoring system

\* User-friendly Streamlit interface



\## Technologies Used



\* Python

\* Streamlit

\* NLP-based evaluation (basic scoring)



\## How to Run



```bash

pip install streamlit

python -m streamlit run app.py

```
